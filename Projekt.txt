Ziel: Erstellen Sie ein Programm zur Verwaltung von Kontakten, das verschiedene Datentypen in Python verwendet, darunter Zahlen, Strings, Listen, Tupel, Sets und Dictionaries.

Funktionen des Programms:
Kontakt hinzufügen:

Jeder Kontakt besteht aus einem Namen, einer Telefonnummer und einer E-Mail-Adresse.
Der Name wird als String gespeichert.
Die Telefonnummer als String (zum Beispiel "123-456-7890").
Die E-Mail-Adresse als String.
Kontakte anzeigen:

Die Kontakte werden in einer Liste von Dictionaries gespeichert.
Jeder Dictionary repräsentiert einen Kontakt und hat Schlüssel wie 'Name', 'Telefon', und 'Email'.
Kontakte suchen:

Sie können nach Namen suchen und entsprechende Kontaktdaten finden.
Hier kann ein Set verwendet werden, um eine Liste von Suchergebnissen zu speichern und doppelte Ergebnisse zu vermeiden.
Kontakt löschen:

Löschen eines Kontakts durch Namen.
Es wird überprüft, ob der Kontakt existiert, bevor er gelöscht wird.
Kontakt aktualisieren:

Aktualisieren von Telefonnummer oder E-Mail-Adresse eines bestehenden Kontakts.
Kontaktliste sortieren:

Die Kontakte können nach Namen oder Telefonnummern sortiert werden.
Hier können Listen und Tupel verwendet werden, um die Sortierung durchzuführen.
